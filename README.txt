Team Task Manager (Full Stack)

This is a full-stack web application where users can create projects, assign tasks, and track progress with role-based access.

-----------------------------------
Features
-----------------------------------
- Signup and Login authentication
- Role-based access (Admin / Member)
- Project creation and management
- Task creation, assignment, and status tracking
- Dashboard showing total, completed, and overdue tasks
- Delete project and tasks

-----------------------------------
Tech Stack
-----------------------------------
Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js

-----------------------------------
How to Run Locally
-----------------------------------
1. Go to backend folder:
   cd backend

2. Install dependencies:
   npm install

3. Start server:
   node server.js

4. Open frontend/index.html in browser
